Dokumentasi API Akinator

API ini menyediakan antarmuka untuk memainkan permainan Akinator. Berikut adalah petunjuk penggunaan API secara urut:

1. Mulai Permainan

Endpoint: /start

Metode: GET

Deskripsi: Memulai permainan Akinator dan mendapatkan pertanyaan pertama.

Contoh Respon:

{
    "question": "Apakah karakter ini manusia?"
}

2. Menjawab Pertanyaan

Endpoint: /answer

Metode: GET

Parameter Query: q (Jawaban, bisa y, n, idk, p, atau pn)

y: Ya atau Benar

n: Tidak atau Salah

idk: Tidak Tahu

p: Mungkin Saja

pn: Mungkin Tidak

Deskripsi: Mengirimkan jawaban untuk pertanyaan yang sedang ditanyakan.

Contoh Respon jika karakter ditemukan:

{
    "name": "Harry Potter",
    "description": "Seorang penyihir yang terkenal dari seri buku Harry Potter."
}

Contoh Respon jika masih ada pertanyaan:

{
    "question": "Apakah karakter ini memiliki kekuatan super?"
}

3. Kembali ke Pertanyaan Sebelumnya

Endpoint: /back

Metode: POST

Deskripsi: Mengembalikan permainan ke pertanyaan sebelumnya.

Contoh Respon:

{
    "question": "Apakah karakter ini seorang penyihir?"
}

4. Mengecualikan Karakter

Endpoint: /exclude

Metode: POST

Deskripsi: Mengecualikan karakter dari hasil permainan dan melanjutkan permainan.

Contoh Respon:

{
    "question": "Apakah karakter ini memiliki teman hewan peliharaan?"
}

5. Melihat Kemajuan Permainan

Endpoint: /progress

Metode: GET

Deskripsi: Mendapatkan informasi tentang kemajuan permainan saat ini.

Contoh Respon:

{
    "progression": 50,
    "step": 5
}

Silakan gunakan endpoint-endpoint ini untuk berinteraksi dengan API Akinator dan nikmati permainan!
